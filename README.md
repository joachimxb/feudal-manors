# Feudal Manors · Fees & Fighting Men (ACKS II)

An interactive reference on knight-fee economics, garrisons, officers, and
domain doctrine for the *Adventurer Conqueror King System* (ACKS II).

**Live page:** https://joachimxb.github.io/feudal-manors/

The whole thing is a single self-contained `index.html` — no build step, no
dependencies. Open it locally in any browser, or view it live at the link above.
